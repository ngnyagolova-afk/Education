Вектори — Демо проекти (C# WinForms)
======================================

4 готови проекта:

1. Часовник        — стартирай: Часовник\bin\Release\net9.0-windows\Часовник.exe
2. Трансформации   — стартирай: Трансформации\bin\Release\net9.0-windows\Трансформации.exe
3. АнимацияТочки   — стартирай: АнимацияТочки\bin\Release\net9.0-windows\АнимацияТочки.exe
4. Билярд          — стартирай: Билярд\bin\Release\net9.0-windows\Билярд.exe

Управление:
  Трансформации  : избери трансформация от левия панел
  АнимацияТочки  : избери режим с бутоните (Ротация / Мащаб / Срязване / Комбо)
  Билярд         : влачи от бялата топка; R = нова игра

За да компилираш наново: dotnet build -c Release
